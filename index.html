Getting started with Luau

Lua vs Luau

What is the difference between Lua and Luau? While Luau is technically derivatived from Lua and using specifically in Roblox, there is not a lot of differences. Performance is the main changes, along-side with more features and more security.

For example, Luau support Compound Assignment. So you can do something like; "y += x + 10" in Luau, but in Lua you have to do like this; "y = y + x + 10". Another example is that Luau has Sleep Function, like Wait(). While Lua does not.

And did you know that Lua, Famously in game development, is also used in Gary's Mod, World of Warcraft, Dota, Angry Birds etc. Where Lua is primarily used as a scripting language due to its ease of embedding, fast execution, and relatively short learning curve, so even you can learn it..

Luau Syntax

In this tutorial we will be focusing on Luau, which is the main scripting language of Roblox. Let's start with:

  • Variable

Assuming you already have some knowledge of Coding, But let's review what exactly is a variable again. A variable is basically a storage for values, and what is a value? It can be a number, string, boolean etc. Which we will get to them later, here is how you can declare a variable: "local hotdog = fridge.hotdog", in this example we start with the keyword local which make it a variable that you can only use locally, such as inside a function, and you also have to declare the variable before you can access it. A global variable is a variable you can use anywhere, just simply remove the local keyword. And in the path fridge.hotdog we access an item by first declare the parent fridge we then assumingly access its child hotdog. With this you now type hotdog instead of fridge.hotdog everytime to access the hotdog.

Rules of thumb when naming Variable is that: A variable name must start with a letter or an underscore character like "_", A variable name cannot start with a number like "1, 2" etc. A variable name can only contain alpha-numeric characters and underscores "a-z, A-Z , 0-9 , and _ " Variable names, or Luau in general are case-sensitive like, age, Age and AGE are three different variables.

Functions and Datatypes

Now that we've learned about Variable, let's start with:

  • function

What is a function? A function is a block of code that you can reuse just how many times you want, here is how you can declare a function: "function sayHello(name)
print("Hello ".. name)
end".
In this example function, we declare it by first typing function, next we assign it a name which in the example: sayHello, and the parenthesis () represent a parameter, and inside the parameter contain a variable: (name). So what is a parameter? A parameter is where you can store variables that a function can use, and to assign a value to the variable you have to call the function. Here is how to call a function: "sayHello("Ryan Gosling")". We simply type the function's name and a parenthesis, which you can pass values to the function. In this case it is a string, what is a string? A string is sequences of any characters like letters, numbers etc. Enclosed in a quotation mark " ", in this case it is "Ryan Gosling".

A string is also a data type, and there are other data types such as:

  • Number(integer are whole number like "1", while float are decimal like "1.2"
  • Boolean(Have only two possible value which are:true or false)
  • table(is an array, allowing you to store multiple values of any data type. You can declare it by using curly brackets: {})
  • nil(absence of a value)

There are other data types like: thread, enums, userdata, vector.. But we're just gonna focus on the basic stuff for now. To wrap up the function, the output of it will be Hello Ryan Gosling".

Conditional and Operators

Now let's jump into the next section: Conditional Statement, what is a conditional statement? A conditional statement uses if, else, elseif, and boolean values to determine whether a condition is met, it is also a block level code like function. For instance, here is how you can declare a condition: if age == 13 then
print("This person is 13")
elseif age == 18 and isMale == true then
print("This person is 18, and is a male")
else
print("This person is neither 13 or 18, and is not a male")
end
in this example, you can see the usage of if, else, elseif and boolean values, also the keyword then, and. Adding an and allows you to add more conditions before processing. In the first condition you can see that: if age == 13 then, the "==" is an equal to operator, it is to check if the age is equal to 13 or not which is different from "=", that is, to assign value.

There are also other operators such as:

  • ~= (not equal to sign, example: x ~= y, x is not equal to y)
  • < (less than sign, for example 6 < 7, 6 is less than 7)
  • > (greater than sign, an example 7 > 6, 7 is more than 6)
  • <= (less than or equal to sign: x <= 10, x is less than or equal to 10)
  • >= (more than or equal to sign: x >= 10, x is more than or equal to 10)

In the second condition you can see the usage of elseif, which if the first if statement does not met, then it will move on to the second if statement, which have 2 condition: elseif age == 18 and isMale == true then, whether the age is equal to 18 AND whether the variable isMale is equal to true for it to print the output. Now there is also the keyword "or", which require one of the condition to be true instead of both: elseif age == 18 or isMale == true then.
In this example, if the first condition(age == 18) does not met, but the second condition(isMale == true) does met then it will still process. You can also combine or, and. Like an ice cream must have a cone and ice cream, but the flavor of the ice cream can be vanilla or cholocate. There is also a not keyword which is pretty much the same as ~= or not equal to. And the last condition is just else, you can use else when the results does not met any of the if and elseif statements.

Loops and Dictionaries

What are loops? A loop is a way to repeat a block of code multiple times without rewriting it, Let’s start with the simplest loop: the for loop. Think of it as a counter. Here’s an example:
for i = 1, 5 do
print("Number: " .. i)
end
In this example, the loop starts with i at 1 and counts up to 5, printing "Number: 1", "Number: 2", and so on. You can adjust the start and end numbers. For instance: for i = 10, 1, -1 do counts backward from 10 to 1. The third number, -1, is the step, here it is decrement by 1 like 10 to 9 then to 8, but you can change to -2, -3 etc.

Another kind of loop is the while loop. It runs as long as a condition is true. I'll Give you an example:
local num = 1
while num <= 5 do
print("Count: " .. num)
num = num + 1
end
This loop starts with num = 1 and increases num by 1 until it reaches 5. It’s great when you don’t know beforehand how many times you’ll need to repeat something, and you can also stop the loop by using: break.

Next, we have the repeat loop. It’s similar to a while loop but always runs the code block at least once. Example:
local x = 0
repeat
print("x is " .. x)
x = x + 1
until x > 5
Here, the loop runs until x becomes greater than 5. Even if the condition is false initially, the loop will run at least once. You can also add a wait() or task.wait() to any loops, just put the duration(second) in the parenthesis.

Now let’s talk about dictionaries. In Luau, dictionaries are tables with keys and values, like a digital filing cabinet. Here’s how you declare one:
local fruits = {apple = "red", banana = "yellow", grape = "purple"}
Here, apple, banana, and grape are keys(keys are similar to variables), and their colors are the values. To access a value, just use its key: print(fruits["apple"]), which will print red, the [] you see is to get its value.

You can also loop through dictionaries using the pairs function. For example:
for fruit, color in pairs(fruits) do
print(fruit .. " is " .. color)
end
This loop prints all the keys and their assigned values, like "apple is red".

Did you know you can mix data types in dictionaries? For example:
local mixedTable = {name = "Ryan Gosling", age = 42, isActor = true}
Here, name is a string, age is a number, and isActor is a boolean. Dictionaries are pretty useful right?

You can combine loops and dictionaries for more practical usages. For instance okay, let’s calculate the total of some scores:
local scores = {LouBloom = 10, TylerDurden = 15, officerK = 12}
local total = 0
for _, score in pairs(scores) do
total = total + score
end
print("Total score: " .. total)
This example adds up all the scores and prints "Total score: 37". That wraps up the Loops and Dictionaries.

Local, Server and Module script

Since we're reaching the end of this Tutorial, i'll teach you about the different script objects uses in Roblox Studio for Luau. A local script runs in the client side, a client is basically your device, computer, phones etc. Anything that runs in the local script will only happen to the person's device it is running on.

A server script, like the name suggest, it runs on the server. Which mean whatever is running in the server script will be replicates to everyone that are in the server, like if color of a part is changed through the server script, everyone will see the change, but not the same case with local script.

A module script is a reusable script that allows you to store functions, variables, and logic which can be shared across multiple scripts. Instead of duplicating code, you can create a module script and "require" it in your local or server scripts. which can be useful to prevent from making too many scripts that runs the same purpose.

For example, you could use a module script to define shared utilities like mathematical functions, or to manage game data that's accessed by both the client and server.
